# Smart Access Portal 🚀

A modern Role-Based Access Management System built using Angular, Node.js, Express.js, and MongoDB.

## 📌 Features

* Secure Login Authentication
* Role-Based Access Control
* Admin & General User Dashboard
* User Management System
* Analytics Dashboard with Charts
* Dark Mode Support
* Responsive UI Design
* REST API Integration
* Protected Routes using Auth Guards
* MongoDB Database Integration

---

## 🛠️ Tech Stack

### Frontend

* Angular 12+
* TypeScript
* HTML5
* CSS3
* Chart.js

### Backend

* Node.js
* Express.js

## ⚙️ Installation

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
ng serve
```

Frontend runs on:

```txt
http://localhost:4200
```

Backend runs on:

```txt
http://localhost:5000
```
